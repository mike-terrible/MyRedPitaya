boot.bif - файл описания структуры загрузчика boot.bin

boot.bin - загрузчик на SD-карте. Состоит из обычно из трёх разделов. Структура описывается в текстовом файле boot.bif 

img:{[bootloader] /home/mike/bif/fsbl/executable.elf /home/mike/bif/led_blinker.bit /home/mike/bif/u-boot.elf }

u-boot.elf - это так нужно переименовать файл u-boot после компиляции.
